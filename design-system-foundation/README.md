# Meshy AI 设计系统基础 (Foundation)

## 📋 说明

本目录保留了Meshy AI设计系统的理论基础，包含基于45个CSS文件深度分析得出的核心设计规范。

**注意**: 这是理论化设计系统的保留版本，完整的基于真实源文件的新版本请查看 [`../design-system/`](../design-system/)。

## 🎨 核心设计基础

### 设计令牌系统
- **[`01-design-tokens.md`](./01-design-tokens.md)** - 完整的设计令牌系统 (2707个CSS变量)
- 包含颜色、字体、间距、阴影、动画等完整的设计基础

### 颜色系统基础
- **[`02-color-system.md`](./02-color-system.md)** - 颜色系统规范 (785种颜色)
- 品牌色验证 (#C5F955, #FF97C2等)
- OKLCH现代颜色空间支持

### 字体排版系统
- **[`03-typography.md`](./03-typography.md)** - 字体排版系统
- 4种字体栈 (Inter, Karma, Inter Tight, Figtree)
- 响应式字体系统

### 间距布局系统
- **[`04-spacing.md`](./04-spacing.md)** - 间距与布局系统
- 8px网格系统
- 27个间距值
- 响应式断点系统
