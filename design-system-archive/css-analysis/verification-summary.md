## 🔍 识别的问题与差异

### 1. 组件命名不一致 ⚠️
**问题**: 组件类名使用CSS Modules或hash命名
- 实际: `button-htraTX`, `btnContainer-NdEnEK`
- 预期: `btn`, `card`, `nav` 等语义化命名

**影响**: 组件识别困难，需要调整搜索策略

### 2. 设计系统混合使用 ⚠️
**发现**: 三套设计系统并存
1. Tailwind CSS (utility-first)
2. Semi Design (component library)
3. 自定义品牌系统

**挑战**:
- 设计令牌分散
- 维护复杂度高
- 一致性风险

### 3. 颜色系统细微差异 ⚠️
**发现**: 品牌粉色存在两个版本
- 预期: #FF3E8F
- 实际更多使用: #FF97C2

**需要验证**: 哪个是当前标准色
