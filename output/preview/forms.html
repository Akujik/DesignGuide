表单布局

常用的表单布局模式和分组方式

垂直布局

水平布局

.form-horizontal .form-group { display: flex; gap: var(--space-4); align-items: center; } .form-horizontal label { min-width: 80px; font-weight: var(--font-weight-medium); } .form-horizontal input { flex: 1; }

内联布局

.form-inline { display: flex; gap: var(--space-3); align-items: center; } .form-inline input { flex: 1; } .form-inline button { padding: var(--space-3) var(--space-6); background-color: var(--color-primary); color: var(--color-text-inverse); border: none; border-radius: var(--radius-md); font-weight: var(--font-weight-medium); cursor: pointer; }
