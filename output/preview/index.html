使用指南

如何使用这个设计系统来构建一致的用户界面。

1. 集成样式

在你的项目中引入 CSS 文件：

<link rel="stylesheet" href="design-system/assets/css/styles.css">
<link rel="stylesheet" href="design-system/assets/css/components.css">

2. 使用组件

使用预定义的 HTML 结构和 CSS 类：

<button class="button button-primary">
  点击我
</button>

3. 自定义设计令牌

通过 CSS 变量自定义设计系统：

:root {
  --color-accent-base: #your-color;
  --font-sans: "Your Font", sans-serif;
}

4. 响应式设计

所有组件都支持响应式设计，使用断点系统：

@media (max-width: 768px) {
  .component {
    /* 移动端样式 */
  }
}
