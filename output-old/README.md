# Meshy AI 设计系统分析报告

## 概述

本设计系统基于对 Meshy AI 官方网站的深入分析，完整提取了其设计规范和组件库，形成了一套可直接使用的设计系统文档和代码。

## 分析结果

### 📊 数据统计
- **分析页面**: Meshy AI 官网完整页面
- **提取的CSS文件**: 9个样式文件，总大小约 1MB
- **分析的设计令牌**: 50+ 个核心变量
- **识别的组件**: 5大类主要组件
- **生成文档**: 10个详细文档
- **预览页面**: 9个交互式预览

### 🎯 核心发现

#### 1. 设计哲学
- **深色优先**: 采用 #181818 作为主背景色，营造专业创作环境
- **品牌突出**: 使用 #C5F955 (绿色) 作为主品牌色，具有极高的识别度
- **层次分明**: 通过颜色深浅、字重、间距建立清晰的视觉层次
- **现代简约**: 整体设计语言简洁现代，去除冗余装饰

#### 2. 技术特点
- **CSS变量**: 大量使用CSS自定义属性，便于主题切换和维护
- **响应式**: 采用移动优先的响应式设计策略
- **性能优化**: 分层加载样式表，优化关键渲染路径
- **可访问性**: 符合WCAG AA标准的色彩对比度

## 核心设计令牌

### 🎨 颜色系统
```css
:root {
  /* 背景色系 */
  --color-bg-base: #181818;      /* 主背景 */
  --color-bg-sub: #1e1e1e;       /* 次要背景 */
  --color-bg-shade: #303030;     /* 深色阴影 */
  --color-bg-border: #3f3f3f;    /* 边框 */

  /* 文本色系 */
  --color-label-title: #ffffff;  /* 主要标题 */
  --color-label-base: #dedede;   /* 正文文本 */
  --color-label-soft: #9b9b9b;   /* 次要文本 */

  /* 品牌色系 */
  --color-accent-base: #C5F955;  /* 主要品牌绿 */
  --color-accent-support-base: #FF3E8F; /* 支持品红 */
}
```

### 🔤 字体系统
- **主字体**: Inter, system-ui, sans-serif
- **字重范围**: 300-700 (Light, Normal, Medium, Semibold, Bold)
- **字体大小**: 0.75rem - 3rem (12px - 48px)
- **行高**: 1 - 2 (无额外行高到很宽松)

### 📏 间距系统
- **基础单位**: 0.25rem (4px)
- **缩放系列**: 4px, 8px, 12px, 16px, 20px, 24px, 32px
- **容器宽度**: 256px - 768px (16rem - 48px)

## 主要组件

### 1. 按钮组件 (Buttons)
- **主要按钮**: 渐变绿色背景，悬浮上移动画
- **次要按钮**: 半透明背景，毛玻璃效果
- **幽灵按钮**: 透明背景，边框样式
- **图标按钮**: 32x32px 正方形按钮

### 2. 卡片组件 (Cards)
- **标准卡片**: 基础信息展示容器
- **模型卡片**: 包含预览图的3D模型展示
- **功能卡片**: 突出特定功能的卡片
- **统计卡片**: 数据展示专用卡片

### 3. 表单组件 (Forms)
- **输入框**: 圆角设计，焦点状态突出
- **文本域**: 可调整大小的多行输入
- **选择器**: 下拉菜单样式
- **复选框/单选框**: 自定义样式
- **开关**: iOS风格的切换开关

### 4. 导航组件 (Navigation)
- **头部导航**: 粘性定位，毛玻璃背景
- **侧边栏**: 固定定位，层次清晰
- **面包屑**: 路径指示导航
- **标签页**: 内容切换导航
- **分页**: 页码导航组件

## 动效系统

### 时间函数
- `ease-out`: cubic-bezier(0, 0, 0.2, 1) - 标准退出动画
- `ease-in-out`: cubic-bezier(0.4, 0, 0.2, 1) - 平滑过渡
- `ease-bounce`: cubic-bezier(0.68, -0.55, 0.265, 1.55) - 弹跳效果

### 动画时长
- **快速**: 180ms - 状态切换
- **标准**: 300ms - 颜色变化
- **慢速**: 600ms - 布局变化
- **很慢**: 1000ms - 页面进入

## 使用方法

### 1. 浏览预览
打开 `output/preview/index.html` 查看完整的设计系统预览

### 2. 复制代码
- 从 `output/design-system/` 获取详细的设计规范
- 从 `output/component-library/` 获取组件使用说明
- 直接复制 `output/preview/` 中的HTML和CSS代码

### 3. 集成到项目
```css
/* 引入样式 */
<link rel="stylesheet" href="styles.css">

/* 使用组件 */
<button class="button button-primary">主要按钮</button>
<div class="card">卡片内容</div>
```
