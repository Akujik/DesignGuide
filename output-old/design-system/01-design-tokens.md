# Meshy AI 设计系统 - 设计令牌总览

## 概述

Meshy AI 采用基于深色主题的现代化设计系统，融合科技感与活力的品牌色彩，为AI 3D模型生成工具提供专业且吸引人的用户界面。

## 核心设计原则

- **深色优先**: 以深色背景为基础，减少视觉疲劳
- **品牌突出**: 使用鲜明的绿色作为主要品牌色
- **层次清晰**: 通过颜色、字重、间距建立清晰的视觉层次
- **响应式**: 适配多种设备和屏幕尺寸
- **可访问性**: 确保良好的对比度和可用性

## 使用指南

### 1. CSS 变量引用
```css
.my-component {
  background-color: var(--color-bg-base);
  color: var(--color-label-base);
  border-radius: var(--radius-lg);
}
```

### 2. 设计工具集成
- 将这些设计令牌同步到 Figma、Sketch 等设计工具
- 确保设计与开发的一致性

### 3. 主题切换
系统支持深色/浅色主题切换，通过修改 CSS 变量值实现：

```css
[data-theme="light"] {
  --color-bg-base: #ffffff;
  --color-label-base: #181818;
}
```
