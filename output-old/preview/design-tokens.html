设计令牌总览

Meshy AI 采用基于深色主题的现代化设计系统，融合科技感与活力的品牌色彩，为AI 3D模型生成工具提供专业且吸引人的用户界面。

核心设计原则

深色优先

以深色背景为基础，减少视觉疲劳，营造专业的创作环境。

品牌突出

使用鲜明的绿色作为主要品牌色，增强品牌识别度。

层次清晰

通过颜色、字重、间距建立清晰的视觉层次。

响应式

适配多种设备和屏幕尺寸，保证一致的用户体验。

CSS 变量使用示例

使用设计令牌创建组件

通过CSS变量，您可以轻松创建符合设计系统规范的组件。

CSS 代码 
:root {
  /* 颜色变量 */
  --color-bg-sub: #1e1e1e;
  --color-label-title: #ffffff;
  --color-accent-base: #C5F955;

  /* 间距变量 */
  --padding-xl: 1.25rem;
  --margin-lg: 1rem;

  /* 圆角变量 */
  --radius-lg: 0.5rem;
}

.component {
  background-color: var(--color-bg-sub);
  border: 1px solid var(--color-bg-border);
  border-radius: var(--radius-lg);
  padding: var(--padding-xl);
  margin-bottom: var(--margin-lg);
}

.button-primary {
  background: var(--color-accent-base);
  color: var(--color-bg-base);
  padding: var(--padding-md) var(--padding-lg);
  border-radius: var(--radius-lg);
}

使用指南

如何使用 Meshy AI 设计系统

1. 引入样式文件

将设计系统的 CSS 文件引入到您的项目中，包含所有的设计令牌和组件样式。

2. 使用设计令牌

在您的 CSS 中使用 var() 函数引用设计令牌，确保样式的一致性。

3. 复用组件

直接复制组件的 HTML 结构和对应的类名，快速构建界面。

4. 自定义扩展

基于现有设计令牌创建自定义组件，保持设计一致性。

最佳实践

  • 优先使用设计令牌而不是硬编码的值
  • 保持组件的语义化，便于维护和理解
  • 遵循响应式设计原则，适配多种设备
  • 注意可访问性，确保良好的对比度和键盘导航
  • 定期更新设计令牌，与设计系统保持同步
