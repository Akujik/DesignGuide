按钮组件

Meshy AI 的按钮组件具有现代的设计风格，支持多种状态和尺寸，通过渐变背景、边框和阴影效果创造出层次感强的视觉体验。

主要按钮 (Primary)

HTML 代码 
<button class="button button-primary">Primary Button</button>
<button class="button button-primary" disabled>Disabled State</button>

次要按钮 (Secondary)

HTML 代码 
<button class="button button-secondary">Secondary Button</button>
<button class="button button-secondary" disabled>Disabled State</button>

幽灵按钮 (Ghost)

HTML 代码 
<button class="button button-ghost">幽灵按钮</button>
<button class="button button-ghost" disabled>禁用状态</button>

图标按钮 (Icon)

HTML 代码 
<button class="button button-icon">
  <svg width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
    <path d="M3 12h18m-9-9v18"/>
  </svg>
</button>

交互演示

点击按钮查看交互效果

交互状态说明

  • 悬浮: 按钮上移并显示阴影
  • 激活: 按钮回到原位，保持阴影
  • 禁用: 降低透明度，禁用交互
  • 加载: 显示旋转动画，禁用点击
